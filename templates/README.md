# Templates

This folder provides reusable templates for daily note-taking and planning.

## Included Templates

- [daily-notes-template.md](daily-notes-template.md): a structured daily reflection template
- future templates can include syllabus drafts, feedback forms, and quiz outlines

## Summary File Convention

Daily participant summaries should be stored in the summaries folder and named using the pattern <name-date>.md.
Example: alex-2026-07-07.md
