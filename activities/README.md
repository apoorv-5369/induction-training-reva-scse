# Activities

This folder is intended for activity prompts, instructional design exercises, and examples that faculty can work on during the creation phase of the daily session.

## Activity Types

- self-study synthesis tasks
- syllabus redesign prompts
- lab activity design challenges
- classroom discussion planning
- AI-resilient assessment design

## Suggested Workflow

1. Choose a topic or teaching problem.
2. Create or revise an artifact.
3. Ask the evaluative agent for feedback.
4. Refine the output before the quiz phase.
