# Quizzes

This folder contains the scaffolding for the assimilation quiz experience.

## Purpose

The quiz should not be a generic knowledge check. It should be informed by the day's session history, especially the feedback from the formative evaluator.

## Intended Characteristics

- personalized to the faculty member's difficulties
- focused on conceptual understanding and pedagogical choices
- adaptive in difficulty and scope
- designed to reveal whether the faculty member can transfer ideas into practice
