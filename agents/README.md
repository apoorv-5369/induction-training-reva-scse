# Agents

This folder contains the foundational specifications for the four specialist agents that power the daily faculty induction experience.

## Agent Inventory

- [CS Ed Expert Agent](cs-ed-expert-agent.md) — the primary facilitator and Socratic guide
- [Formative Evaluator Agent](formative-evaluator-agent.md) — the pedagogy critic and mentor
- [Assimilation Quiz Engine](assimilation-quiz-engine.md) — the adaptive end-of-session assessor
- [Logging & Ledger Agent](logging-ledger-agent.md) — the hidden system memory and reflection layer

## Design Principles

Each agent should:

- support the faculty member without taking over their thinking
- encourage reflection rather than answer delivery
- make the daily workflow feel personal and adaptive
- preserve a record of progress for later review
